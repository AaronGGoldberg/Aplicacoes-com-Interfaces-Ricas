# CRUD Angular 21 + PrimeNG 21

## 📌 Descrição

Aplicação CRUD desenvolvida com Angular 21 e PrimeNG 21.

## 🚀 Funcionalidades

* Criar produtos
* Listar produtos
* Editar produtos
* Remover produtos
* Visualizar detalhes

## 🧱 Tecnologias

* Angular 21
* PrimeNG 21
* TypeScript

## 📦 Modelo

Produto:

* nome (string)
* preco (number)
* ativo (boolean)

## ▶️ Como rodar

```bash
npm install
ng serve
```
